# PyBeam
PyBeam is Python bindings for mcray library.

### Authors:
   Dolgikh Konstantin

### Features
 All mcray features in new Python interface

### Applications

 Targeting algorithms research

### Installation
- Install pybind11: `pip3 install pybind11` or `make pybind11`
- Build code:
<pre><code>cd mcray
make pybeam
</code></pre>
